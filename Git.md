## 在命令行中输入以下命令

```shell
git config --global user.name "你的名字"
git config --global user.email "Your 邮箱"
```

> 还可以输入以下命令进行查看

```shell
git config --global user.name
git config --global user.email
```



## Linux一些基本命令

```shell
1. mkdir xxx 新建文件夹
2. vi xx.txt 新建/编辑文件
	1. 输入i进入编辑模式
	2. ESC + ：+wq 保存并退出
	3. ESC + : + q! 不保存退出
3. cd xxx 进入xxx目录
4. cd .. 返回上级目录
5. ls 列出当前文件夹中的所有文件
6. pwd 显示当前目录
7. cat x.txt 显示文件内容
8. clear 清屏
```
